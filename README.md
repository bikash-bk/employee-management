# Employee Management System (MERN Stack)

A simple Employee Management System built using:

- **MongoDB** (database)
- **Node.js + Express** (backend REST API)
- **React + Axios + React Router** (frontend UI)

## Features

- Add Employee  
- Edit Employee  
- Delete Employee  
- View Employee List  
- Search Employee  
- Full CRUD with MongoDB  
- Clean UI (Custom CSS)
