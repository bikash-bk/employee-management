# Employee Management System (MERN Stack)

A simple **Employee Management System** built using:

- **MongoDB** (database)
- **Node.js + Express** (backend REST API)
- **React + Axios + React Router** (frontend UI)

This project includes:

✔ Add Employee  
✔ Edit Employee  
✔ Delete Employee  
✔ View Employee List  
✔ Search Employee  
✔ Full CRUD with MongoDB  
✔ Clean UI (custom CSS)

---


steps to download and install project 

**BACKEND**
1️⃣ Navigate to backend
     cd backend
2️⃣ Install dependencies
    npm install
3️⃣ Create .env file inside backend folder
    PORT=5000
MONGO_URI= your_mongo_connection_string_here
4️⃣ Start the backend  
    nodemon server.js   or node server.js



**⚙️ Frontend Setup (React)**

1️⃣ Navigate to frontend
    cd ../frontend
2️⃣ Install dependencies
    npm install

3️⃣ Create .env file in frontend
    VITE_API_URL=http://localhost:5000/api
    
4️⃣ Start frontend
     npm run dev
